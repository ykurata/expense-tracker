# Expense Tracker 

This is an application to track your income and expenses monthly base. Users can see their total savings in chart. 

## Build with

* Backend
    * Node
    * Express
    * Postgres 

* Frontend
    * React
    * Material-UI


##  Features

* Register, login, and logout 
* Create income
* Create expenses
* Add categories 
* See total savings 
* Add profile image
